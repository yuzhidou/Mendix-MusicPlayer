## MusicPlayer

Mendix版本: 9.18.2

##### 项目简介:
使用HTMLSnippet与纳流JS实现开发的一个音乐播放的完整Demo

可以动态上传音乐,歌词,封面图片等

##### 初始化步骤:
1.打开Music页面点击初始化数据

2.点击详情页上传音乐
